## Fonctionnement

1. S’assurer que le fichier `operations.txt` se trouve dans le même dossier que l’exécutable.
2. Compiler puis exécuter le programme.
3. Le programme lit chaque ligne du fichier `operations.txt` et exécute les opérations sur l’arbre binaire de recherche :
   * `I,d` : insère la valeur `d` dans l’arbre
   * `S,d` : supprime la valeur `d` de l’arbre
   * `A` : affiche l’arbre niveau par niveau
   * `H` : affiche la hauteur de l’arbre
   * `G,d` : affiche les ascendants du nœud de valeur `d`
4. Les résultats de chaque opération (affichage de l’arbre, hauteur, ascendants, messages d’erreur, etc.) sont montrés directement dans la console.
